## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

## Customization

- **Personal Info:** Edit `src/pages/index.astro`.
- **Blog Posts:** Add Markdown files to `src/content/blog/`.
- **Layout/Styling:** Edit `src/layouts/Layout.astro` and `src/styles/global.css`.
